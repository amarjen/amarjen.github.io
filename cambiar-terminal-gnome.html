Como cambiar la Terminal por defecto de gnome

Date mié 03 abril 2013 Tags gnome

Resulta que hace unos días decidí probar el terminal Guake y cada vez que lo uso me gusta más, sin embargo, siempre que pulsaba Ctrl + Alt + T, se abría la antigua terminal que gnome trae de serie.

Para cambiar la configuración de gnome abrir una terminal y ejecutar el siguiente comando:

gsettings set org.gnome.desktop.default-applications.terminal exec 'guake'

Ahora siempre aparecerá guake cuando invoquemos la terminal, y recuerda para enconderla y volverla a llamar con F12
